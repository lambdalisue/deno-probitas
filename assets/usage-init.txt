probitas init - Initialize a new Probitas project

Usage: probitas init [options]

Generates the following files:
  deno.jsonc                     Deno config with probitas settings (if not exists)
  scenarios/example.scenario.ts  Sample scenario

If deno.json or deno.jsonc already exists, adds probitas configuration to it.

Options:
  -h, --help     Show help message
  --force        Overwrite existing files

Global Options:
  -q, --quiet    Suppress console.log/warn/info/debug (show errors only)
  -v, --verbose  Allow console.log/info/warn (suppress debug only)
  -d, --debug    Allow all console output (no suppression)

Environment Variables:
  PROBITAS_CONFIG  Path to config file

Examples:
  probitas init           Initialize new project
  probitas init --force   Overwrite existing files
